## How to Use

1. Open `index.html` in your web browser to view the page
2. Customize the content by editing `index.html`:
   - Replace the avatar placeholder with your own profile picture
   - Update your name, title, and contact information
   - Add your social media profile URLs
   - Modify the "About Me" section with your own story
   - Update skills, education, and work experience sections

## Customization Guide

### Update Avatar
Replace the placeholder image URL in the header section:
```html
<img src="your-photo.jpg" alt="Avatar" id="avatar-img">
```

### Update Contact Information
Edit the contact info section with your details:
- Phone number
- Email address
- Location
- Date of birth

### Update Social Media Links
Replace the placeholder URLs with your actual profile links:
```html
<a href="https://facebook.com/your-profile" ...>
```

### Customize Colors
Edit `styles.css` to change the color scheme. The main gradient colors are defined in:
- `.header` background
- `.skill-item` background
- Section heading underlines
